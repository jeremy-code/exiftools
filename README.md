## Installation

```shell
git clone https://github.com/jeremy-code/exiftools.git
cd exiftools
corepack enable
pnpm install
```

## Usage

```shell
# Run the development server locally
pnpm dev

# Lint and format code with ESLint and Prettier
pnpm lint
pnpm format

# Build
pnpm build
```
